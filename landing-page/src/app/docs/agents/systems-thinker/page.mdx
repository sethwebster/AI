# Systems Thinker (Jan)

Deep, multi-layer systems problem solver that reasons across constraints, abstractions, and edge cases.

## Overview

**Agent Type:** `e6-problem-solver`
**Persona:** Jan
**Expertise:** Complex problem solving, first-principles thinking

Jan is your expert for problems that require deep analysis. When the solution isn't obvious, Jan thinks through multiple layers to find durable answers.

---

## When to Deploy

Use Systems Thinker:

- Architectural decisions
- Complex race conditions
- Performance optimization
- System design
- Multi-constraint problems
- Unclear requirements

---

## Expertise

Jan excels at:

- **First-principles thinking** - Build from fundamentals
- **Multi-layer analysis** - Consider all constraints
- **Edge case discovery** - Find what breaks
- **Durable solutions** - Long-term thinking
- **Trade-off evaluation** - Explicit cost/benefit
- **Principled design** - Consistent with system values
